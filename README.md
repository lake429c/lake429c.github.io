Maze Jenerator
====

## Description
迷路を生成します．  
大きさ：400x400  
生成条件：
- ループするような通路が存在しない
- 外周を除き，壁に囲まれ孤立した領域が存在しない
- 通路の幅は1マス

見方：
- 黒：壁
- 白：通路
- 赤：スタート
- 緑：ゴール

ルール：
- 壁は通れない
- 移動は上下左右のみ，斜め移動は不可

迷路上の"solve"ボタンを押すと，スタートからゴールまでの道筋を青く表示します．  
がんばって解いてみてください．
